wireless Topology Mode

Ad hoc mode=This is when the device connect wirelessly in a peer to peer (P2P) manner without using APs or router.

Infracstructure Mode=This is when wireless clients inteeconnect via a wireless router or AP ,such as in WLANs APs connect to the network Infracstructure using the wired distribution system,such as ethernet.

Tethering=This is state of host posting and the smart device to internet ,other devices can associate and authenticate with the smart phone to use the internet connection.
