Basic Service Set And Extended Service Set

Basic Service Set=It consist of a single AP interconnecting all associated wireless clients,if the client move out of BSS Network coverage it will no longer communicate and also has Basic serice set identifier(BSSID).

Extended Service Set=This is a type of a service set that is made up of twoo BSS (which can be considered as Multiple AP) and ESS is identified by SSID and each BSS is identified by BSSID.
